# ly_math
Lyceum Yandex Math project
### Requirements

* python 3.x.x (tested on 3.7.9)

### Installation
```
git clone https://github.com/MlsDmitry/ly_math.git
cd ly_math
pip3 install -r requirements
```

### Startup application
```
python3 startup.py
```
